## :alarm_clock:このプロジェクトについて

幼馴染好きのあなたにおくる、作業効率向上 App の決定版（自称）
App を起動すれば、幼馴染と雑談をしながら作業をやることに。またスマホやネットサーフィンをしていると𠮟ってくれます。
幼馴染成分不足の方、作業が進まない人、よくわからないけど面白そうと思う方におすすめです。<br><br>


## 🔥開発者たちの想い

幼馴染は圧倒的王道ヒロイン、いいね？<br><br>
